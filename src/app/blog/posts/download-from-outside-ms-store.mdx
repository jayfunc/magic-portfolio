1. Visit https://store.rg-adguard.net/
1. 访问 https://store.rg-adguard.net/

2. Type MS Store app link (e.g. https://apps.microsoft.com/detail/9p1wcd1p597r) in the link input area
2. 在链接输入框中输入微软应用商店应用链接（例如 https://apps.microsoft.com/detail/9p1wcd1p597r）

3. Select Retail from the drop-down list
3. 从下拉列表中选择 “零售版”

4. Click the check mark
4. 点击勾选标记

5. Select the largest installation package in the resulting list to download and install. If you fail to install, try to install the dependency packages first.
5. 在结果列表中选择最大的安装包进行下载和安装。如果安装失败，请先尝试安装依赖包。
