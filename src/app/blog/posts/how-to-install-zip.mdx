## Preparation

Be sure that you have already enabled developer mode. To do that, you can follow the steps below:

### 1. Open "Settings", select "System", and then go to "Developer options".

![](https://github.com/jayfunc/BetterLyrics/blob/dev/Sideloadly/image-0.png?raw=true)

### 2. Enable "Developer mode" and allow local PowerShell scripts to run.

![](https://github.com/jayfunc/BetterLyrics/blob/dev/Sideloadly/image-1.png?raw=true)

Now everything is ready.

## Steps

### Step 1

Extract the downloaded .zip file, right-click "install.ps1", and select "Run with PowerShell".

![](https://github.com/jayfunc/BetterLyrics/blob/dev/Sideloadly/image-2.png?raw=true)

### Step 2

Press "Enter" to continue and agree to the pop-up window.

![](https://github.com/jayfunc/BetterLyrics/blob/dev/Sideloadly/image-3.png?raw=true)

### Step 3

Enter "Y" to install the cert.

![](https://github.com/jayfunc/BetterLyrics/blob/dev/Sideloadly/image-5.png?raw=true)

### Step 4

Now everything is ready.

![](https://github.com/jayfunc/BetterLyrics/blob/dev/Sideloadly/image-6.png?raw=true)

> If the installation fails with an older version already installed, try uninstalling the old version and reinstalling.
